# shortid-cli

Leismore ShortID Generator (CLI) is a command-line tool for generating new Leismore ShortIDs.

## Installation

`npm install -g @leismore/shortid-cli`

## Usage

`lm-shortid`, or

`npx @leismore/shortid-cli`
